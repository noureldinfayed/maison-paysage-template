# Maison Paysage — Premium Landscaping Next.js Template

A cinematic, high-end landscaping and outdoor design template built with Next.js, Three.js, GSAP and Tailwind CSS.

## What's Included
- Full Next.js 15+ source code (TypeScript)
- Three.js animated hero scene
- GSAP scroll animations throughout
- Contact form with Resend integration
- Mobile responsive design
- Complete documentation

## Quick Start
```bash
npm install
cp .env.example .env.local
# Fill in your values in .env.local
npm run dev
```

## Tech Stack
- Next.js 15 + TypeScript
- Tailwind CSS
- Three.js (hero animation)
- GSAP + ScrollTrigger
- Framer Motion
- Resend (contact form)
